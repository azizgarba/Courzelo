Test formula

Calculate test grade

A provided formula is used to calculate the test grade for each student in the final/quiz test class based on their performance in both the final test and quiz test components:

  • N: number of questions = 10
  • C: number of correct answers
  • w: weight assigned to each question = 2

Module average

Calculate the module evaluation.

a provided formula to calculate the module average based on the attendance grade, quiz test grade, and final test grade

  • F be the score achieved in the final test.
  • Q be the score achieved in the quiz test.
  • be the score achieved by attendance.
  • wF​ be the weight assigned to the final test = 60%
  • wQ be the weight assigned to the quiz test =40%
  • G=(F×wF)+(Q×wQ)

Final evaluation

Calculate the final evaluation.

A provided formula is used to calculate the overall average grade for each student based on their module averages :

  • F be the coeffecient of all modules.
  • Qi be the module grade of each module.
  • G=Qi/F
