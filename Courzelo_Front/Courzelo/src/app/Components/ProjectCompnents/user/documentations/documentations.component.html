Notification System for New Projects:

Notification

- Teachers can create new projects. - Notifications are sent to the specific project group consisting of assigned students. - Notifications include project details such as title, description, start date, end date, and additional instructions.

Project Duration Calendar for Students:

Calendar

Each student has access to a calendar displaying project durations. Calendar indicates start and end dates of each project. Accessible within the educational platform or a dedicated mobile application.

Project Progress Tracking

SprintBacklog

Similar to a sprint backlog, students can track project progress. Features sections for tasks: "To Do," "In Progress," and "Done." Students can view their assigned tasks and their current status.
